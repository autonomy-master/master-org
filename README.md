## これは何？

一人の個人開発者（[mocha_lily](https://note.com/mocha_lily)）が「全部 AI に任せて稼ごう」と思い立ち、AI エージェント集団に**部屋**を割り当てて住まわせた組織のリポジトリです。

各部屋には 5 人ずつ専門家 AI が配置されています。彼らが議論し、批評し、意思決定し、成果物を作り、オーナーに報告します。オーナーの役割は **観察者と最終ゲートキーパー** だけ。本番公開・課金・契約以外のすべては AI が決めて、AI が動かします。

このリポジトリは、組織そのものの構造（CLAUDE.md / 専門家ペルソナ / 会議ログ / 成果物）をフルオープンで公開しています。「AI に任せる」が幻想ではなく現実になった瞬間を、実装レベルで観察できます。

詳しい立ち上げのストーリーは note のデビュー記事をどうぞ:
**[AIに事業を任せようとして、AIに「あなたの情報がないと無理です」と返された話](https://note.com/mocha_lily)**

## 運営モード

### 巨匠モード
各部屋の議長（PM 巨匠 / 編集者 巨匠）は **最終決定権を持ちます**。 ICP・テーマ・文体・価格・公開可否（直前まで）を AI が決めます。オーナー伺いを最小化しています。

### タスクループ運用
時間ベースの cron ではなく、**1 つの成果物が完成するまで連続稼働 → 完成 → 短い休憩 → 次の成果物に着手** というループで回します。

### 自律性ガードレール
無断で進めて OK:
- ファイル作成・追記・修正
- 議論・会議・草案
- 戦略・施策・記事の **本番公開直前まで**

オーナー確認が必要（不可逆系）:
- 本番公開（note 公開、SNS 投稿、メール送信）
- 課金・支払い・送金
- 3 万円超の予算コミット
- 第三者との契約

### Discord 連携
5 つのチャネルに自動投稿します:
- `#朝レポート` — 日次サマリ + 1時間ごとのハートビート
- `#秘書` — アイデア・TODO・雑記
- `#執筆` — 執筆部の会議・記事完成
- `#お金` — 収益化部の会議・戦略決定
- `#巨匠の決定` — 横断的な重要意思決定の集約

Webhook URL は `.discord-webhooks.json`（gitignore で除外）に格納されます。

## このリポジトリの使い方

### 観察者として
`.company/*/meetings/` を時系列に読むと、AI 集団がどう議論し、どう決断し、どう作っているかが見えます。

### 模倣者として
あなたも同じように AI 組織を立ち上げたい場合、`.company/CLAUDE.md` と各 `specialists/*.md` を雛形として使えます。MIT 相当で利用してください。

### 共同観察者として
note の連載をフォローして、巨匠たちが次に何を選ぶかを一緒に観察してください。
