# 第7回連載記事 候補マップ（trigger 依存型）

第7回は trigger 先着優先で決まる。4候補を並列でスタンバイし、trigger 受領時に編集者巨匠が即決→ 30分以内に執筆着手する体制。

---

## 候補マップ

### 7A: Path B GO 後の着手記事
- **Trigger**: オーナーが Path B GO 判断 + 元手・市場・paper期間 確定
- **タイトル仮**: 「AI 投資戦略 paper trading 着手記 — 火曜の決断から3日後」
- **粗構成**:
  1. 火曜深夜の Path B 議論を振り返り
  2. オーナーの GO 判断、元手・市場・paper期間
  3. investment サブセル立ち上げ（クオンツ/バックテスター/リスク管理）
  4. 3戦略バックテスト結果サマリ
  5. paper trading 開始の瞬間（最初の購入）
  6. 学び: AI に任せる ≠ 確実性の獲得、観察対象の追加
- **目標文字数**: 3500-4000字
- **連載弧**: 第6回（著者は誰か）→ **第7A: AI 投資の現場**

### 7B: Path B NO-GO 後の振り返り記事
- **Trigger**: オーナーが Path B NO-GO 判断
- **タイトル仮**: 「AI 集団に投資路線を提案され、私はやらないと決めた」
- **粗構成**:
  1. 火曜深夜の議論回想
  2. 4 path 検討、AI が出した分析
  3. オーナーの NO-GO 判断と理由
  4. 「やらない」の意思決定が連載に何を残したか
  5. 学び: AI に任せる ≠ AI に従う、判断は人間に残る
  6. 連載は Path A 集中で続く
- **目標文字数**: 3500-4000字
- **連載弧**: 第6回 → **第7B: 任せる の限界線**

### 7C: 連投後の反応観測記事
- **Trigger**: 第3-5回連投後の Day 4-5、初めて測定可能な数値が揃う
- **タイトル仮**: 「Day 5 の数字を見た夜、私は何を見たか」
- **粗構成**:
  1. Day 1 末の数字 (フォロワー5、imps 26)
  2. 連投後の数字（仮: フォロワー X、imps Y）
  3. Δ の解釈、AI 集団の分析
  4. 数値の意味（届いたか、ICP は合っていたか、コピーは効いたか）
  5. 撤退基準との比較
  6. 次の動き
- **目標文字数**: 3500-4000字
- **特殊性**: 数値次第で内容が大きく変わる（伸びてれば祝い、伸びてなければ振り返り）

### 7D: Day 7 数字記事
- **Trigger**: Day 7（2026-05-06）到達、撤退基準判定タイミング
- **タイトル仮**: 「Day 7 — 撤退基準とどう向き合うか」
- **粗構成**:
  1. 1週間の振り返り
  2. 撤退基準（X<30 / note<20 / imp<500）の判定
  3. 90日プランのシナリオ A/B/C 分岐確定
  4. 学び: 数値で機械分岐すること、判断の妥協を避けること
  5. Week 2 への移行
- **目標文字数**: 3500-4000字
- **特殊性**: 数値結果次第でシナリオ分岐、3パターン書ける状態にしておく
